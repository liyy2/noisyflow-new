NoisyFlow

Source-side differentially private optimal transport from private client cohorts to an assumed-public target reference distribution.

NoisyFlow trains client-side DP flow generators and DP neural optimal-transport maps, then composes the released parameters on the server to synthesize reference-aligned pseudo-data for downstream classifiers.

Method

A three-stage differentially private optimal-transport framework.

The paper separates the release of private source summaries, the estimation of source-to-reference transport, and server-side synthesis so each stage can be accounted for under record-level differential privacy.

01

DP source generator

Each client trains a label-conditional flow-matching generator with DP-SGD, producing pseudo-source samples from its local distribution.

02

DP neural OT

An ICNN-parameterized convex potential learns source-to-reference transport, with privacy handled by Option A, B, or C depending on the training signal.

03

Server aggregation

The server composes the released DP artifacts to generate reference-aligned pseudo-data, aggregates it across clients, and trains a classifier.

Evaluation

Evaluation endpoints for utility, alignment, privacy accounting, and attack risk.

Utility is compared across Ref-train-only, transported ỹ-only, and Ref-train+ỹ training regimes; untransported x̃-only ablations isolate the effect of optimal transport.

Ref-train-only

Classifier trained only on labeled samples from the reference training split.

ỹ-only

Classifier trained only on transported, reference-aligned pseudo-data.

Ref-train+ỹ

Classifier trained on both reference labels and transported pseudo-data.

Reference distribution

Shared target-side distribution used for OT; target test remains held out.

Privacy accounting

Per-client (ε,δ)-DP is computed from DP-SGD clipping and Gaussian noise; noised label priors are optional and used only for class sampling.

Membership-inference tests

Stage-level MIA uses per-example flow-matching losses and OT-derived statistics; ROC AUC near 0.5 indicates chance-level membership signal.

Results

Transported pseudo-data improves over few-shot reference-only baselines.

Across PBMC, BrainSCOPE, CAMELYON17-WILDS, and PAMAP2, transported pseudo-data improves target accuracy relative to small reference-only baselines; the paper reports both ỹ-only and Ref-train+ỹ regimes.
