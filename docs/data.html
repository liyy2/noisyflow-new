Additional Builders

The repository also includes builders for toy Gaussians, generic federated cell tables, CellOT Statefate, SciPlex3, and 4i proteomics. These are supported code paths, not additional paper-result claims unless paired with a publication config.

Reproduction Checklist

  • Create the raw or prepared dataset artifact under datasets/<dataset-name>/.
  • Run the matching configs/publication/*.yaml file from the repository root.
  • Record the seed, preparation command, upstream data version, and generated result JSON or plot.
  • For downloaded public datasets, keep cache and split metadata with the local experiment record.
